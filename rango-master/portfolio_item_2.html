Análisis de Riesgo

Criterios de Decisión

Beneficio Monetario o VAN Esperado

Se acepta el proyecto si BME(VANE) es mayor de cero, y para proyectos múltiples se elige el mayor BME. Consideramos que el BME(VANE) es una variable aleatoria (b). En este criterio no se tiene en cuenta el riesgo del proyecto, es decir, el nivel de variación de b.

Equivalente de certidumbre VC (EC)

Cuando a(s) tiende a cero, los riesgos son más altos. La tasa de descuento, i, es la que la empresa puede asignar sin riesgo. El criterio de aceptación es si el VC(EC)>0, y para proyectos múltiples de inversión el orden de selección es de mayor VC (EC).

El tanto incrementado con prima de riesgo VC (PR)

Tasa de descuento con prima de riesgo VC (PR)

Este criterio incluye de manera indirecta el riesgo del proyecto, al incluir la tasa de descuento, i, que la empresa considera sin riesgo: una prima por riesgo del proyecto. Ejemplo: " i' = i + p"

Esperanza - Desviación Típica E

d: Parámetro indicativo de la aversión al riesgo asignado por el decisor, d>0

o: (VAN) es la desviación típica de la variable.

El decisor acepta un proyecto de inversión aislado si E>0, y frente a varios proyectos son preferidos los de mayor E.

Coeficiente de Variación

o: Desviación típica

R: Rendimiento esperado

Es un indicador que nos permite clasificar para la selección de inversiones, es decir, ver el nivel del riesgo. El decisor inclinará su decisión para las alternativas que tengan menor CV

Distribución de probabilidades del VAN

Para el caso de una variable aleatoria discreta, como el VAN, es importante evaluar la probabilidad de cada evento aleatorio, y para una variable aleatoria continua es necesario evaluar la función de probabilidad, ya que el evento tiene lugar en un dominio continuo. Es decir, debemos evaluar una distribución de probabilidades. En los siguientes ejercicios aplicaremos estos conceptos.     • Simulación de Monte Carlo en la evaluación de proyectos

    Ante el problema de no poder resolver situaciones de inversión con métodos analíticos, se debe desarrollar la distribución del VAN mediante la simulación, aplicando modelos de Monte Carlo.

    La aplicación de la simulación por computadora no se limita a simular un fenómeno físico, pues se ha utilizado para tomar decisiones de inversión antes de que se ejecuten realmente. El enfoque general es asignar una distribución de probabilidad, ya sea subjetiva u objetiva, a cada factor desconocido, com binando todas las distribuciones en una distribución de probabilidad para la rentabilidad del proyecto conjunto; de esta manera se puede obtener la distribución del VAN resultante.

    Como recordará, la demanda unitaria (X) en el proyecto de fabricación de ladrillos fue una de las variables aleatorias del problema. Podemos conocer el valor exacto de esta variable sólo después de im plementar el proyecto.

    En este punto del libro estudiamos la aplicación de software, como Crystal Ball, para analizar riesgos del proyecto, con la fundamentación teórica del modelo de Monte Carlo. La secuencia lógica para aplicar correctamente esta simulación, que nos permite determinar el riesgo del proyecto, sería:

    • Paso 1: Identificar las variables críticas que afectan el valor del VAN.
    • Paso 2: Plantear correctamente la ecuación financiera para calcular el VAN del proyecto y las variables que se relacionan.
    • Paso 3: Clasificar las variables en dos grupos: los parámetros cuyos valores se conocen y las variables aleatorias para las que no se especifican sus valores.
    • Paso 4: Definir distribuciones para las variables aleatorias.
    • Paso 5: Realizar el muestreo Monte Carlo, aplicando el software Crystal Ball.
    • Paso 6: Calcular los parámetros de distribución y sus representaciones gráficas de los resultados de la simulación.
